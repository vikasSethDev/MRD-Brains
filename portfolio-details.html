ML.NET Deep Tutorial with Example: Sentiment Analysis Project

What is ML.NET?
ML.NET is an open-source, cross-platform machine learning framework developed by Microsoft. It empowers .NET developers to integrate custom AI and ML models directly into their C#, F#, or VB.NET applications—no Python or R required.

ML.NET Architecture & Key Concepts

  • MLContext: The starting point for all ML.NET operations, managing data, training, and prediction.
  • IDataView: ML.NET’s flexible, high-performance data pipeline abstraction.
  • Pipeline (EstimatorChain): A series of transformers and a trainer that define the steps for preparing and training models.
  • ITransformer (Model): The result of fitting the pipeline—ready to make predictions.

End-to-End Example: Sentiment Analysis Project

Let’s build a sentiment analysis tool that classifies text reviews as positive or negative.

  1. Initialize Your .NET Core Console Project
    Install ML.NET with: dotnet add package Microsoft.ML
  2. Prepare Your Data
    Example data sentiment_data.txt format:
    Wow... Loved this place. 1
    Crust is not good. 0
    Not tasty and the texture was just nasty. 0
    The food was delicious! 1
    Would not recommend. 0
  3. Define the Data Structures
    public class SentimentData
{
    [LoadColumn(0)]
    public string Text { get; set; }
    [LoadColumn(1), ColumnName("Label")]
    public bool Sentiment { get; set; }
}

public class SentimentPrediction
{
    [ColumnName("PredictedLabel")]
    public bool Prediction { get; set; }
    public float Probability { get; set; }
    public float Score { get; set; }
}
  4. Load & Split the Data
    var ctx = new MLContext();
var data = ctx.Data.LoadFromTextFile<SentimentData>("sentiment_data.txt", hasHeader:false, separatorChar:'\t');
var split = ctx.Data.TrainTestSplit(data, testFraction:0.2);
  5. Build and Train the Pipeline
    var pipeline = ctx.Transforms.Text
    .FeaturizeText("Features", nameof(SentimentData.Text))
    .Append(ctx.BinaryClassification.Trainers.SdcaLogisticRegression("Label", "Features"));

var model = pipeline.Fit(split.TrainSet);
  6. Evaluate the Model
    var predictions = model.Transform(split.TestSet);
var metrics = ctx.BinaryClassification.Evaluate(predictions);
Console.WriteLine($"Accuracy: {metrics.Accuracy:P2}");
Console.WriteLine($"AUC: {metrics.AreaUnderRocCurve:P2}");
  7. Make Real-Time Predictions
    var predEngine = ctx.Model.CreatePredictionEngine<SentimentData, SentimentPrediction>(model);
var sample = new SentimentData { Text = "The food was fantastic, will come again!" };
var result = predEngine.Predict(sample);
Console.WriteLine($"Prediction: {(result.Prediction ? "Positive" : "Negative")} - Probability: {result.Probability:P2}");

Going Deeper: ML.NET Capabilities

ML.NET supports advanced tasks including:

  • Multiclass Classification (e.g., topic or intent detection)
  • Regression (e.g., price or sales forecasting)
  • Clustering (e.g., customer segmentation)
  • Recommendation Systems
  • Time Series Forecasting
  • Object/Image Recognition (via ONNX/TensorFlow integration)
  • Anomaly Detection

You can further extend ML.NET by importing external models or using AutoML for automated model training and tuning.
